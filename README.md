======================

<!-- Description -->

## Installation

### Setting up
Please run the following commands:
- bundle install
- npm install
- bundle exec rails db:create
- bundle exec rails db:migrate
- bundle exec rails db:seed

### Running the app
Please run the following commands in seperated tabs:
- rails s
- npm start
